HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the web. It defines the meaning and structure of web content and is a core technology of the World Wide Web alongside CSS and JavaScript.

History

HTML was invented by Tim Berners-Lee in 1991 while he was working at CERN. The first version was a simple set of tags to describe web documents. Over time, it evolved through versions HTML 2.0, 3.2, 4.01, and finally HTML5, which became the modern standard developed by the W3C and WHATWG.

Features

HTML provides elements for text formatting, multimedia embedding, links, lists, tables, forms, and semantic structure. HTML5 introduced new elements like <section>, <article>, <nav>, <video>, and <canvas>, making it more powerful for building web apps and multimedia content.

Usage

HTML is used by every website on the internet. It forms the skeleton of web pages and works alongside CSS (for styling) and JavaScript (for interactivity). Developers use HTML to build websites, applications, email templates, and more.
